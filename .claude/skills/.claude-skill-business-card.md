# 大地工業電子名片管理 Skill

## 概述

這個 Skill 協助你快速管理電子名片系統：
- ✅ 一鍵新增同事名片
- ✅ 修改現有名片資訊
- ✅ 自動 Git 推送
- ✅ 生成 NFC 卡片網址

## 快速命令

### 新增同事

**Windows 用戶：**
```bash
cd D:\OneDrive\文件\claude\電子名片
powershell -ExecutionPolicy Bypass -File add-member.ps1
```

**Mac / Linux 用戶：**
```bash
cd /path/to/business-card
chmod +x add-member.sh
./add-member.sh
```

### 修改現有名片

1. 編輯 `team/[名字]/index.html`
2. 修改內容（名字、職稱、郵件等）
3. 推送到 GitHub：
   ```bash
   git add team/[名字]/
   git commit -m "Update [name]'s business card"
   git push origin main
   ```
4. 等待 30 秒自動更新

### 查看所有同事

打開 `employees.csv`，查看：
- 所有同事的名字和聯絡方式
- 資料夾名稱和網址
- 輕鬆更新資訊

## 常見工作流程

## NFC 卡片寫入指南

### 準備工作
- **Android**：下載 "NFC Tools" app（免費）
- **iOS**：使用 Shortcuts app 或第三方 NFC 工具

### 步驟

1. **開啟 NFC Tools**
2. **進入 WRITE 標籤**
3. **Add record → URL**
4. **輸入網址**
   ```
   https://DDchJason.github.io/business-card/team/x.wang/
   ```
5. **Write/Execute**
6. **貼卡片**（保持靜止 2-3 秒）

### 測試
用另一支手機掃描卡片 → 自動打開網頁名片 ✅

## 常見問題

**Q: 名片多久會更新？**  
A: 30 秒到 2 分鐘內自動部署

**Q: 修改了名片，訪客還是看到舊版？**  
A: 訪客需要硬刷新（Ctrl+Shift+R 或 Cmd+Shift+R）

A: 可以改資料夾名稱，但 NFC 卡片必須重新寫入

**Q: 可以改 logo 嗎？**  
A: 不建議，有特殊需求請聯絡 IT

**Q: 如何查看所有同事？**  
A: 打開 `employees.csv`

## 🔍 推送檢查流程

**每次推送 Git 後，必須執行以下檢查：**

### 推送前
1. 檢查所有 HTML 結構
2. **確認 Logo 路徑為 `logo.svg`（同目錄，不是 `../logo.svg`）**
3. 驗證所有內容已替換
4. 確認 vCard 和頁脚已移除

### 推送後
1. 等待 1-2 分鐘 GitHub Pages 部署
2. **打開每個員工名片頁面檢查：**
   - Logo 圖片是否正常顯示（不是文字）
   - 頁面標題是否正確
   - 內容是否正確顯示（沒有跑掉）
   - 聯絡方式是否完整
   - LINE @dadi999 是否顯示

### ⚠️ Logo 檢查最關鍵
- Logo 檔案必須在 `team/[名字]/` 目錄中
- HTML 中路徑必須是 `src="logo.svg"`（相同目錄）
- **不是** `src="../logo.svg"`（上一級目錄）
